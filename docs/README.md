## 作用
阻止浏览器跳转链接时使用应用内浏览器（CustomTab）

理论上支持所有使用 CustomTab 的浏览器

### 已测试
| 浏览器 | 版本 |
| --- | --- |
| Via | 6.9.0 |
| Chrome | 143.0.7499.192 |
| Edge | 143.0.3650.139 |

<details>
<summary><strong>制作原因</strong></summary>

Via 在版本 6.9.0 适配了链接应用内打开 导致想要使用浏览器内置的下载管理器等页面需要再点击一个按钮跳转到 Via 非常繁琐

本模块用于移除此特性。

</details>
